# Egypt Real Estate Price Prediction

A Machine Learning project for analyzing and predicting real estate prices in Egypt using Python and regression models.

---

# Overview

This project builds a machine learning pipeline to predict real estate prices based on property features such as size, location, bedrooms, and bathrooms.

---

# Libraries Used

numpy, pandas, matplotlib, seaborn, scikit-learn, xgboost, joblib

---

# Installation

```bash
pip install numpy pandas matplotlib seaborn scikit-learn xgboost joblib
```

---

# System Architecture

Data -> Cleaning -> Feature Engineering -> Model Training -> Evaluation -> Prediction
